前端通过HTTP请求（Axios），把数据发给后端接口；

后端通过URL + HTTP方法匹配接口，返回数据；

双方通过HTTP协议，用JSON作为数据格式进行通信



# 一、Vue3 + Axios 是怎么调用后端接口的？

Vue3和Axios可以通过发送HTTP请求给后端

```javascript
axios.get('http://localhost:8080/user/1')
```

这段代码表示游览器向后端发送一个HTTP请求

```javascript
请求方式：GET
请求地址：http://localhost:8080/user/1
```

**一个完整的Vue3+Axios示例：**

前端：

```javascript
import axios from 'axios'

axios.get('http://localhost:8080/api/user/1')
  .then(response => {
    console.log(response.data)
  })
```

后端：

```typescript
@RestController
@RequestMapping("/api/user")
public class UserController {

    @GetMapping("/{id}")
    public User getUser(@PathVariable Integer id) {
        return new User(id, "Tom", 18);
    }
}
```

后端响应前端的请求后，返回JSON数据

```javascript
{
  "id": 1,
  "name": "Tom",
  "age": 18
}
```

Axios拿到数据以后，先解析JSON，然后把解析后的数据放入response.data



# 二、后端需要做的事情

后端根本不关心前端是谁。

**后端只做三件事：**

1. 监听端口

```javascript
localhost:8080
```

* 根据URL + 请求方式 匹配接口

```javascript
GET /api/user/1
```

* 返回数据



# 三、前端和后端是通过HTTP协议进行通信

一次请求的完整过程

```javascript
Vue3
 ↓
Axios
 ↓
HTTP 请求
 ↓
浏览器
 ↓
网络
 ↓
Spring Boot
```

HTTP请求包含三部分：

1. 请求行

```javascript
GET /api/user/1 HTTP/1.1
```

* 请求头

```javascript
Content-Type: application/json
Authorization: Bearer xxx
```

* 请求头

```javascript
{
  "name": "Tom",
  "age": 18
}
```



# 四、前后端通过JSON进行数据交换

前端发送JSON

```javascript
axios.post('/api/user', {
  name: 'Tom',
  age: 18
})
```

游览器实际发送的是：

```javascript
{
  "name": "Tom",
  "age": 18
}
```

后端接收JSON

```typescript
@PostMapping("/api/user")
public void save(@RequestBody User user) {
    System.out.println(user.getName());
}
```

由Spring Boot 自动完成JSON和Java对象互相转换
